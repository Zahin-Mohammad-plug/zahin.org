# zahin.org
My personal portfolio website

## Version 2 (Current)
Built with Next.js, TypeScript, and Tailwind CSS. Features:
- Modern, responsive design
- Multiple sections: About, Projects, Stack, Passions
- Dark/light theme support
- Component-based architecture

## Development
```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

## Deployment
The site is configured for static export and deploys automatically to GitHub Pages via GitHub Actions.

## Legacy Version
The original v1 site (simple HTML/CSS/JS) is preserved in the `/v1` directory and accessible at `https://zahin.org/v1/`
